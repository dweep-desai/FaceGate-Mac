# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in FaceGate, please report it privately via email to **dweep.design@gmail.com**.

Please **do not** open a public GitHub issue for security vulnerabilities.

I will acknowledge your report within 48 hours and work on a fix as soon as possible. You will be credited for the discovery unless you request otherwise.

## Supported Versions

Only the latest release on the [releases page](https://github.com/dweep-desai/FaceGate-Mac/releases) receives security updates.
